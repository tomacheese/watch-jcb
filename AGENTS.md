# AGENTS.md

## 目的
このドキュメントは、AI エージェント共通の作業方針を定義します。

## 基本方針
- 会話は日本語で行う。
- コード内のコメントは日本語で記載する。
- エラーメッセージは英語で記載する。
- コミットメッセージは Conventional Commits に従い、日本語で内容を記載する。
- 日本語と英数字の間には半角スペースを入れる。

## 判断記録のルール
- なぜその設計や実装を選んだのか、判断プロセスを記録する。
- 代替案を検討し、不採用の理由を明記する。
- 前提条件や不確実な点を明示する。

## 開発手順
1. プロジェクト構造と既存のコード規約を理解する。
2. `pnpm install` で依存関係をセットアップする。
3. 実装を行い、`pnpm lint` および `pnpm test` で品質を確認する。

## セキュリティ / 機密情報
- 設定ファイル (`data/config.json`) やログに API キーや Webhook URL などの機密情報を出力・コミットしない。

## リポジトリ固有
- このリポジトリは JCB カードのキャンペーン情報を Discord に通知するツールです。
- TypeScript と pnpm を使用しています。
- スクレイピングには `cheerio` を、文字コード変換には `iconv-lite` (エンコーディングは `windows-31j`) を使用しています。
